# Reference Articles

Store links and notes about reference articles here.

Do not paste full competitor articles into this folder.

Recommended format for each reference note:

- source URL
- target keyword or topic
- why it ranks or why it is useful
- content structure summary in your own words
- missing angle Sever Studio could cover
- entities covered
- internal linking observations
- CTA observations

Never copy, translate, or closely paraphrase competitor content.
