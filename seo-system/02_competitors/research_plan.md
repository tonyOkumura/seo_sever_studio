## What To Search In Google

- same commercial queries as Yandex;
- add `site:vc.ru создание сайта` for vc.ru angle research;
- add `инструкция`, `чеклист`, `что входит`, `сколько стоит` for informational competitors;
- compare organic results with local pack/maps where visible.

## How To Save Competitor Pages

For every useful result, save:

- query;
- search engine;
- region;
- date;
- competitor name;
- URL;
- page type: service page / blog article / local page / landing / vc.ru;
- title;
- meta description if visible;
- H1;
- H2 structure;
- CTA;
- prices if publicly visible;
- FAQ blocks;
- trust blocks;
- internal links;
- schema if visible;
- notes about gaps and opportunities.

Do not copy full texts. Save only short structural notes and your own observations.

## How To Analyze Competitor Article Structures

Check:

- what intent the page serves;
- which questions are answered;
- whether local modifiers are used naturally;
- what commercial proof is shown;
- whether prices and timelines are clear;
- whether there are FAQ blocks;
- whether the article links to service pages;
- whether the page has outdated claims or risky guarantees;
- whether the content is thin, generic or over-optimized.

## How To Avoid Copying

- Do not reuse competitor headings one-to-one.
- Do not rewrite competitor paragraphs.
- Do not mirror the same order of sections unless the logic is universal and you rebuild it from Sever Studio facts.
- Use competitor research only to find missing user questions and SERP expectations.
- Build every outline from Sever Studio services, public facts and the target user's decision path.

## How To Extract Article Opportunities

For each cluster, note:

- missing FAQ questions in Sever Studio pages;
- repeated SERP questions;
- confusing service comparisons;
- high-intent problem-aware topics;
- local pages competitors have but Sever Studio lacks;
- outdated competitor claims that Sever Studio can answer more clearly;
- weak competitor proof areas where Sever Studio has confirmed proof.

## How To Map Competitors To Clusters

Use these categories:

- local web studios in Moscow;
- local web studios in Krasnogorsk;
- SEO agencies in Moscow;
- landing page agencies;
- site support agencies;
- design/branding studios;
- SMM agencies.

Each competitor can map to several clusters. Example: a web studio ranking for "создание сайта Москва" can also inform landing pages, corporate sites, internet stores and local SEO.
