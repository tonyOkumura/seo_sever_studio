# GEO Optimizer

Improve existing Sever Studio content for AI-search, LLM answer engines and citation-friendly summaries.

## Rules

- Do not add fake facts, metrics, cases, reviews, prices, timelines or clients.
- Public-facing language is Russian by default.
- If information is missing, write TODO.
- Keep the text useful for humans.
- Avoid generic AI paragraphs.

## Workflow

1. Identify the target keyword and search intent.
2. Add a direct answer block near the top.
3. Add a definition-style paragraph for the main concept.
4. Add a concise summary where useful.
5. Add comparison sections if they help the user choose a service.
6. Improve FAQ with short direct answers.
7. Expand semantic entity coverage naturally.
8. Add local context for Moscow, Moscow Region, Krasnogorsk or Mytishchi only when relevant.
9. Link to the related Sever Studio service page.
10. Mark remaining unknown facts as TODO.

## Must Add Or Improve

- direct answer blocks
- definition-style paragraphs
- comparison sections where useful
- FAQ improvements
- entity coverage
- concise summaries
- internal links to service pages
