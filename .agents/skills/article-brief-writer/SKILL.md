# Article Brief Writer

Create a detailed SEO/GEO brief before writing any full article for Sever Studio.

## Context

- Brand: Sever Studio.
- Website: https://sever.studio/
- Public-facing content: Russian by default unless English is explicitly requested.
- Search focus: Yandex, Google Russia/CIS, local SEO for Moscow, Moscow Region, Krasnogorsk and Mytishchi.
- Main services: website development, landing pages, corporate sites, internet stores, SEO, support, site refinement, audits, Yandex Direct, VK/SMM, branding/design, copywriting/rewriting.

## Rules

- Do not write the full article.
- Do not invent facts, fake case studies, fake metrics, fake reviews, fake clients, fake prices or fake timelines.
- If information is missing, write TODO.
- Do not default to SaaS/MVP/startup product design unless the user explicitly asks or the topic is archive/future.
- Do not copy competitor structures.
- Use semantic coverage, not keyword repetition.

## Workflow

1. Identify the target keyword, cluster and search intent.
2. Confirm the related Sever Studio service page or write TODO URL.
3. List confirmed facts and mark their source in internal notes.
4. Mark missing facts as TODO.
5. Define the audience and funnel stage.
6. Choose a practical article angle.
7. Build an H1/H2/H3 structure.
8. Add semantic entities and local modifiers where relevant.
9. Add FAQ questions for Yandex snippets and AI-search answers.
10. Add internal link suggestions and CTA.
11. Draft meta title and meta description.

## Required Output

- target keyword
- search intent
- audience
- funnel stage
- article angle
- H1/H2/H3 structure
- semantic entities
- FAQ questions
- internal link suggestions
- CTA
- meta title
- meta description
- notes for writer
